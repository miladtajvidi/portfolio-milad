Unsupervised Clustering

MSc Project

  • Conducted unsupervised clustering on a dataset to identify clusters without prior labels, exploring multiple clustering algorithms
  • Performed EDA with feature distribution visualizations, identifying anomalies and guiding feature selection
  • Applied preprocessing steps like power transformation for scaling and removed low-variance features to enhance clustering accuracy
  • Tested clustering models including K-Means, Gaussian Mixture Model (GMM), and Bayesian Gaussian Mixture Model (BGMM), using ARI as the evaluation metric
  • Achieved the highest Adjusted Rand Index (ARI) score with BGMM, indicating optimal model performance for cluster differentiation
  • Outlined future directions including custom algorithms like Expectation Maximization and deeper statistical analyses for feature selection
